Pour nouveau projet : 
Créer un compte GitHub
Créer un repository
Y mettre les fichiers avec un premier upload
Dans settings > GitHub pages, on peut voir si elles sont en ligne
Dans appli GitHub faire clone repository : préciser le répertoire en local (1er onglet - localpath) et le répertoire distant (3ème onglet - url)
On peut ensuite modifier en local et pusher dans la branche adéquat
